## 기능구현 목록
### 입력 기능
- 비상근무 배정 시작 월, 요일 입력받기
- 평일 근무 순번 입력받기
  - 이름 중복 X
  - 이름 최대 5글자
  - 사람 최소 5명, 최대 35명
- 휴일 근무 순번 입력받기
  - 이름 중복 X
  - 이름 최대 5글자
  - 사람 최소 5명, 최대 35명

### 비상근무 배정 규칙
- 두 종류의 날짜 존재, 평일과 휴일(주말 및 법정공휴일)
- 평일에서 휴일 넘어갈 때, 중복된다면
  - 그 다음 휴일 근무자와 교대
- 휴일에서 평일 넘어갈 때, 중복된다면
  - 그 다음 평일 근무자와 교대
- 2월은 항상 28일까지

### 출력 기능
- 에러 상황 시, 적절한 문구 출력하기
- 평일이지만, 법정 공휴일인 경우 요일 뒤에 `(휴일)` 표기하기
- 비상 근무표 출력하기
